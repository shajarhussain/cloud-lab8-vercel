## Project Description

This project demonstrates a Next.js frontend deployed on Vercel with serverless API routes, environment variables, middleware, edge runtime, and preview deployments.

## Features

- Custom homepage with student name and enrollment
- Environment variables using `.env.local`
- Serverless GET endpoint: `/api/hello`
- Dynamic API route: `/api/student/[id]`
- POST API endpoint: `/api/submit`
- Edge runtime endpoint: `/api/geo`
- Middleware header and edge log
- Preview deployment through feature branch
